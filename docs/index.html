Jaamo Photo Downloader

Browse, select, and download all daycare photos from the Jaamo parent portal — with EXIF metadata automatically added to every file. Works with any daycare using the Jaamo platform.

Python 3.9+ macOS · Windows · Linux Jaamo platform Open Source

What it does

🔐

Secure login

Logs in with your Jaamo credentials. Optionally saves them locally so you don't have to retype them.

👶

Child selector

Automatically detects all children on your account. Pick one to browse their photos.

🖼️

Gallery view

Photos grouped by date with captions. Scroll through everything in one view.

✅

Flexible selection

Click individual photos, select a whole day, or grab everything at once.

📍

EXIF metadata

Injects date, caption, and GPS location into every downloaded JPEG so your photos library stays organised.

💾

Smart filenames

Files are named SKSG_{name}_{date}_{original}.jpeg — no guessing what's what.

Manual

Start the app

Run python3 app.py. The login screen appears.

(Optional) Set GPS location

Click GPS Location on the login screen to set the daycare's coordinates. These are embedded in the EXIF of every downloaded photo so they show up on a map in your photos app. The coordinates are validated to be within the Netherlands and saved for next time.

Log in

Enter your Jaamo e-mail and password and click Inloggen. Tick Inloggegevens onthouden to save your credentials locally (~/.jaamo_credentials.json, chmod 600) so you don't need to retype them next time.

Select a child

If your account has multiple children a selector screen appears. Click the card for the child whose photos you want to browse. If there is only one child the gallery opens directly.

Browse the gallery

Photos are shown grouped by date. Scroll down to see older photos. Captions appear below each thumbnail where available.

Select photos to download

You have three ways to select:

  • Click a photo — selects or deselects that single photo (blue border = selected).
  • Selecteer dag button — selects or deselects all photos for that date group.
  • Selecteer alles in the toolbar — selects every photo in the gallery.

The toolbar shows how many photos are selected.

Download

Two download options:

  • Download selectie (N) — saves all N selected photos to a folder you choose.
  • Download dag (per date group) — saves all photos from that day to a folder you choose.

Each file is saved as SKSG_{firstname}_{date}_{original}.jpeg, for example SKSG_Emma_15-march-2026_IMG_0558.jpeg.

Refresh if URLs expire

Photo URLs are valid for 10 minutes. If you leave the app open and then try to download, you may get errors. Click Vernieuwen in the toolbar to reload fresh URLs.

Switch child (← Terug)

If you have multiple children, a ← Terug button appears in the toolbar. Click it to go back to the child selector without logging in again.

Note: Downloaded photos have EXIF metadata (date, caption, GPS) injected automatically. The server strips all metadata on upload, so the app re-adds it on every save.

What gets written to each photo

FieldValue
Date / DateTimeOriginalDate shown in the gallery (time set to noon)
Image descriptionCaption from the gallery, if present
GPS coordinatesDaycare location — configurable via GPS Location on the login screen
File modification timeSet to the photo date so Finder/Explorer sorts correctly

Privacy & security

  • Credentials are stored only on your own machine (~/.jaamo_credentials.json) with file permissions set to owner-only (chmod 600).
  • GPS settings are stored in ~/.jaamo_settings.json.
  • No data is sent anywhere other than the Jaamo server you are already authenticated with.
  • The app contains no telemetry, tracking, or analytics of any kind.

Troubleshooting

Login fails

Double-check your email and password on the Jaamo website first. Make sure you are using your daycare's organisation account.

Photos fail to download (403 error)

The photo URLs expire after 10 minutes. Click Vernieuwen to reload fresh URLs, then try the download again.

App doesn't start

Make sure all dependencies are installed: pip install requests beautifulsoup4 Pillow piexif. Python 3.9 or newer is required.

Thumbnails don't load

This can happen on a slow connection. Wait a moment and try scrolling — thumbnails load in the background. Click Vernieuwen if they never appear.
